При первой установке:
1) Разархивировать проект в папку
2) Находясь в папке с прокетом в ее пути сверху
написать "cmd"
3)Далее в консоль ввести команду:
python -m venv venv (создается окружение).
4)Далее нужно его активировать venv\Scripts\activate
5) Далее нужно установить все библиотеки
Запускать команды по очереди и ждать установки:
pip install Django
pip install pandas
pip install scikit-learn
pip install deep-translator
6) После этого ввести команду:
manage.py migrate
7) Создать админа командой:
 python manage.py createsuperuser
 (когда нужно будет ввести пароль, 
ввод не будет отображаться)
8) Запустить приложение:
python manage.py runserver
9) Ввести в браузере хост приложения:
http://127.0.0.1:8000/

При последующих запусках:
1) Ввести в папке с проектом в ее пути сверху команду "cmd"
2)В открывшейся консоли ввести команду для активирования окружения:
venv\Scripts\activate
3)Запустить приложение:
python manage.py runserver
4) Ввести в браузере хост приложения:
http://127.0.0.1:8000/
